────────────────────────────────────────────────────────────────
 STEP 1 — PRE-INSTALL (platform-specific, do this FIRST)
────────────────────────────────────────────────────────────────

  macOS:
    brew install portaudio

  Linux (Ubuntu 20+):
    sudo apt update
    sudo apt install portaudio19-dev python3-pyaudio espeak ffmpeg

  Windows:
    No pre-steps needed. pip handles everything.

────────────────────────────────────────────────────────────────
 STEP 2 — INSTALL PYTHON DEPENDENCIES
────────────────────────────────────────────────────────────────

    pip install -r requirements.txt

  (macOS/Linux use pip3 if pip points to Python 2)

────────────────────────────────────────────────────────────────
 STEP 3 — ADD YOUR OPENROUTER API KEY
────────────────────────────────────────────────────────────────

  Get a free key at:  https://openrouter.ai → Keys → Create Key

  Open  wgm_all_devices.py  and find this line near the top:

    OPENROUTER_API_KEY = "your api key"

  Replace  "your api key"  with your actual key.

  Optional — set your city for weather:

    DEFAULT_CITY = "Mumbai"

────────────────────────────────────────────────────────────────
 STEP 4A — RUN (Voice / Terminal Mode)
────────────────────────────────────────────────────────────────

    python3 wgm_all_devices.py          ← interactive mic mode

  Quick test without mic:

    python3 wgm_all_devices.py "what time is it"
    python3 wgm_all_devices.py "open youtube"
    python3 wgm_all_devices.py "weather in Pune"
    python3 wgm_all_devices.py "latest AI news"
    python3 wgm_all_devices.py "find all pdf files"

────────────────────────────────────────────────────────────────
 STEP 4B — RUN (Web UI Mode — browser interface)
────────────────────────────────────────────────────────────────

  First install Flask if you haven't:

    pip install flask

  Then start the web server:

    python3 web/server.py

  Open in browser:

    Local:    http://localhost:5000
    Network:  http://<your-ip>:5000   ← shown in terminal output

  The web UI works from any browser on the same Wi-Fi network,
  including your phone or tablet.

────────────────────────────────────────────────────────────────
 TROUBLESHOOTING
────────────────────────────────────────────────────────────────

  "No module named flask"
    → pip install flask

  "FileNotFoundError: wgm_all_devices.py"
    → Make sure you run server.py from inside the project root:
       cd WGM-AI-PERSONAL-ASSISTANT
       python3 web/server.py

  "Connection to WGM core failed" in browser
    → Check the terminal where server.py is running for errors
    → Make sure your OpenRouter API key is set

  PyAudio install fails on macOS
    → brew install portaudio   then   pip3 install pyaudio

  PyAudio install fails on Linux
    → sudo apt install portaudio19-dev python3-pyaudio
